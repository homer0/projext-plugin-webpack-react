    ProjextReactPlugin

    This service is in charge of modifying a target Babel and entry settings in order to build React code. It also manages the settings related to the React Hot Loader.

    public register(app: Projext) source

    This is the method called when the plugin is loaded by projext. It just gets the events service and registers the listeners for the reducer events that handles JS rules and target configuration.

    This method gets called when projext reduces the JS rules of a target. It validates the target settings and makes the necessary modifications to the Babel loader configuration.

    public updateTargetConfiguration(currentConfiguration: Object, target: Target): Object source

    This method gets called when projext reduces a target configuration for Wepack. It validates the target settings and if HMR is enabled, it updates the entry setting with the required changes for the React Hot Loader.
